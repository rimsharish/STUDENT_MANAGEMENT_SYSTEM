Project Description

The Student Management System is a simple Python console-based application that helps manage student records. It allows users to register students, update profiles, enroll students in courses, track attendance, enter marks, calculate grades, and generate student reports.

This project is suitable for beginners who are learning Python dictionaries, loops, conditional statements, and basic data management.

Features
1. Student Registration
Add a new student.
Store student ID, name, and age.
Prevent duplicate student IDs.
2. Student Profile Management
View student details.
Update student name.
Update student age.
3. Course Enrollment
Assign a course to a student.
4. Attendance Tracking
Enter total classes and attended classes.
Automatically calculate attendance percentage.
5. Marks Entry & Grade Calculation
Enter student marks.
Automatically assign grades:
Marks Range	Grade
90 - 100	A
80 - 89	B
70 - 79	C
60 - 69	D
Below 60	E
8. Generate Student Report

Displays:

Student ID
Name
Age
Course
Attendance Percentage
Marks
Grade
7. Show All Students

Displays a list of all registered students.

Technologies Used
Python 3
Dictionary Data Structure
Loops (while)
Conditional Statements (if-elif-else)
